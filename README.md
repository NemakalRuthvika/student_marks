# Student Marks Calculator Full Mini Project

## Features
- Calculate total, average, percentage, grade, pass/fail
- Store records in CSV
- Store records in Excel
- Generate PDF report card
- Tkinter GUI
- Mangodb  database integration
- Flask web app
- GitHub + Render deployment ready

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open:

```text
http://127.0.0.1:5000
```

## Run Tkinter GUI

```bash
python gui_tkinter.py
```

## Render Deployment

1. Push project to GitHub.
2. Open Render.
3. New Web Service.
4. Connect GitHub repository.
5. Build Command:

```bash
pip install -r requirements.txt
```

6. Start Command:

```bash
gunicorn app:app
```
